# Mint Standard Library

Pure-Mint standard library modules. Each module provides functions for a specific domain.

**Status:** Empty — stdlib modules will be written during v0.3 development.

## Planned modules

- `collections.mint` — map, filter, reduce, sort, find
- `strings.mint` — split, join, upper, lower, trim, replace
- `math.mint` — abs, sqrt, pow, sin, cos, pi, e (pure Mint implementations)
- `io.mint` — read_file, write_file, file_exists
- `json.mint` — parse, stringify
